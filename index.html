Note

>Everyline of code that runs in java must be inside the class
>the main() Method is required and you will see it in every Java application. any code inside the main() Method will be executed
>system.out.println() inside the main() method used to print the line of text to the screen




Modifying Java program
1.Displaying a single line of text with multiple statements
public class MyClass {
public static void main(String[]) {
system.out.print("Welcome to ");
system.out.println("java programming");
}
}



2.Displaying a multiple lines of text with a single statements
public class MyClass {
public static void main(String[]) {
system.out.println("Welcome\n to\n java\n programming ");
}
}

\n means (new line)
